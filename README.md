# Системы управления библиотекой

Система управления библиотекой — это консольное приложение, написанное на Go. Он позволяет пользователям управлять коллекцией книг библиотеки, брать и возвращать книги, искать книги по названию или автору, а также отображать взятые напрокат книги.

## Функции 
* Добавить в библиотеку новые книги с указанием названия, автора и количества.
* Взять книги в библиотеке.
* Вернуть книги в библиотеку.
* Поиск книг по названию или автору.
* Отображение доступных книг в каталоге библиотеки.
* Отображение книг, которые в настоящее время взяты напрокат.

## Установка
Чтобы использовать систему управления библиотекой, выполните следующие действия:
* Клонируйте репозиторий на свой компьютер:
  `git clone https://github.com/ot4u/LibraryManagementSystem`
* Запустите систему:
  `go run main.go`
